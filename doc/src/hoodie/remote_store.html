pull url

Depending on whether remote is connected (= pulling changes continuously) return a longpoll URL or not. If it is a beginning bootstrap request, do not return a longpoll URL, as we want it to finish right away, even if there are no changes on remote.

function pullUrl() { var since; since = remote.getSinceNr(); if (remote.isConnected() && !isBootstrapping) { return '/_changes?include_docs=true&since=' + since + '&heartbeat=10000&feed=longpoll'; } else { return '/_changes?include_docs=true&since=' + since; } }
